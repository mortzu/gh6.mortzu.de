Access GitHub via IPv6

Unfortunately github.com is only accessible via IPv4. IPv6-only systems cannot use it. GitHub should be available over IPv6. Only GitHub can make that happen, so contact them if IPv6 is important to you. The workaround is to use proxies which are accessible on IPv6. I'm running one proxy for HTTP and SSH.
